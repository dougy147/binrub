Find magic bytes inside any file, and plot Shannon entropy.

```console
$ ./binrub file.bin

DECIMAL         HEXADECIMAL     DESCRIPTION
--------------------------------------------------------------------------------
0               0x0             PDF document
55080           0xD728          Zlib compressed data, No Compression (with preset dictionary)
64692           0xFCB4          Zlib compressed data, No Compression (with preset dictionary)
```

To plot nice entropy graphs in the terminal, install [that](https://github.com/red-data-tools/unicode_plot.rb) gem : `gem install unicode_plot`.
